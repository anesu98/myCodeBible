Data Types

This section talks about the different data types in Javascript. This also includes documentation that discusses the use of them through links to technical documentation

In a Computer Science, Data is anything that is important to the Computer. JavaScript provides 7 different data types.

Undefined

Means a varaible that is hasn't been defined (In other words, has no context in the code).

undefined

Null

Null means nothing. It means you've set the variable to be something, but that something is nothing (has no value)

null

Boolean

A boolean means literally just means 'True' or 'False'

boolean

String

A string refers to the text used as the value of your code.

string

symbol

An immutable, primitive value that is unique

symbol

Number

Number refers to the number used as a value to your code.

number

Object

Stores a lot of key value pairs

object
