Comments

This section shows you how to make comments in Javascript and includes documentation that discusses the use of them through links to helpful articles and technical documentation

In Line Comments

Allows You to add notes at the end of your code. I.e to let a programmer know what the expected outcome would be after a specific line is run

// inline comment

Multi Line Comments

You may need to comment out entire functions of code. Here is how you do it

/* Multi Line Comment */
